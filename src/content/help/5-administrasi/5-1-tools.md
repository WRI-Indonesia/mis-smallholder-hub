**Dashboard Snapshot** — Membuat rekaman data terbaru untuk Main Dashboard dan BMP Dashboard. Jalankan setelah input atau import data dalam jumlah besar agar angka dashboard mengikuti data terkini.

**Riwayat snapshot** — Snapshot lama tetap tersimpan dan dapat dilihat atau diunduh sebagai Excel.
